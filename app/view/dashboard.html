Welcome to RSpactor

New to RSpactor? Follow these first steps to quickly get into the flow:

  1. Create a symlink to get the nifty command line tool (see INSTALL.txt included with the Disk Image)
  2. Check your Preferences (⌘,): Keep up-to-date with the auto-update feature
  3. Kick off: Enter the root path to your ruby project into the box above (⌘+l) and hit 'enter'
